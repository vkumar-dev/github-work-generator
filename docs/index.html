Quick Start

Get up and running in 3 steps

1

Clone the Repository

git clone https://github.com/vkumar-dev/ai-work-generator-framework.git
cd ai-work-generator-framework
2

Install Dependencies

# Install GitHub CLI
sudo apt install gh  # or brew install gh

# Authenticate
gh auth login

# Optional: Python dependencies for GUI
pip install -r requirements.txt
3

Configure & Run

# Copy config template
cp config.example.sh config.sh

# Edit with your settings
nano config.sh

# Run in terminal mode
./workgenerator.sh

# Or with GUI
USE_GUI=true ./workgenerator.sh
