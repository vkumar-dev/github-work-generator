Create workgenerator.sh with: - GitHub analysis using gh cli - AI idea generation with configurable prompts - User decision interface (YES/NO/PASS/IMPROVE) - Task planning and execution - State persistence (.workgenerator_config, .ideas_log.json) - Main loop with configurable rest periods 2. Create config.example.sh with: - GOAL variable - QWEN_BIN (AI CLI command) - REST_SECONDS (default: 14400) - UI options (USE_GUI, WORKGENERATOR_APP_GUI) - Technology preferences 3.
